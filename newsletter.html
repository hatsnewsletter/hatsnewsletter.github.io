The Raspberry Pi RP2040 can now run at 200MH

A new update to the pico-sdk allows the Raspberry Pi RP2040 chip to run at a maximum clock speed of 200MHz instead of the stock 125MHz speed.

This ESP32 Antenna Array Can See WiFi - Jeija

This ESP32 antenna array named "ESPARGOS" was developed by researchers at the Institute of Telecommunications at the University of Stuttgart. It uses a grid of ESP32s to locate where WiFi signals are coming from. There are some very impressive demos in the video of how WiFi signals propagate and reflect.

Sendme - Iroh

A tool for easy file transfer with no accounts, no complications, end-to-end encrypted. Enables you to directly transfer files with the help of a single link.

Calculator Forensics

A very old webpage that explains a method to figure out the chip used inside a calculator without opening it up, exploiting the floating point implementation in the chip. The table of data is a bit outdated, I found out that a calculator model that I’m using has apparently changed ICs since 2007 (shocker!)

Building a laptop from scratch -Byran Huang

Byran designed and built an open-source laptop project that’s seriously impressive. Its chasis is inspired by MacBook and Razer laptops and is built from scratch, it features a 4K AMOLED display, a removable wireless mechanical keyboard, a CNC aluminum chassis, and an RK3588-based motherboard. He tackles PCB design, power management, and firmware development for the build.
