Printing solder to create circuit boards

Somehow a “side-project” in a car build series, this is a 2D gantry that melts solder onto a panel of FR4 to create circuit traces. Seems to be built using a standard 3D printer nozzle and extruder. One downside is that spray glue needs to be applied to the board for the traces to stick.

ErgO Smart ring keyboard+mouse - Sophia Dai

Sophia Dai’s ErgO is a smart, Bluetooth-enabled ring replacing keyboard and mouse in a very small form-factor. It features an IMU, trackball, buttons, and device switching, ideal for everyday tasks and active use. She shares her code and has building instructions on Hackaday.

Bambu Lab A1 Mini automated with automatic plate changer kit

Andre Bar’s DIY kit Swapmod enables a Bambu Lab A1 to become almost fully automated. The printer can swap its own build plate to be able to keep printing indefinitely.

The Czinger 21C Chassis

Czinger Vehicles’ Bio-Logic chassis is designed by optimizing multiple variables together: stiffness, weight, fatigue, crash safety, and manufacturing using AI-driven software. It is additively manufactured with their own patented alloys.

RO1 Advanced Affordable 6 Axis Robot -Standard Bots

RO1 is a advanced, affordable six-axis robot for industrial use, with an 18kg payload and 0.025mm repeatability. It features an integrated 3D camera and a no-code UI for easy automation, requiring no extra controllers, making it simple to integrate into manufacturing setups.
